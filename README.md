Fashion MNIST CNN Classification
Overview
This project implements a 6-layer Convolutional Neural Network (CNN)
to classify images from the Fashion MNIST dataset using Keras in Python and R.

Features
- CNN model with 6 layers
- Image classification into 10 categories
- Predictions on test images

Technologies Used
•  Python (TensorFlow / Keras)
•  R (Keras package)
•  NumPy
•  Matplotlib

How to Run
Ensure that the Python is on Version 3.10 and below in order to install Tensor flow
